# Telegram Printer Agent

Print files remotely by sending them to a Telegram bot.

This project allows you to send PDF files or images to a Telegram bot and have them printed automatically on a Windows computer.

---

## What this project provides

This repository contains the **client-side software only**.

The actual printing service requires **separate activation** to access the Telegram bot and server-side processing.

---

## Supported file types

Supported:
- PDF
- Images (JPG / PNG â†’ automatically converted to PDF)

Not supported:
- Word, Excel
- AutoCAD (DWG)
- Any other file formats

Unsupported files must be exported to PDF before sending.

---

## System requirements

- Windows 10 / 11
- A working printer installed on Windows
- Internet connection
- Telegram account

---

## Installation

1. Download and install the application
2. Select your printer
3. Start the agent (runs in system tray)

The agent can start automatically with Windows.
