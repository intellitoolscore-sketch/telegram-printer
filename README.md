# Telegram Printer Agent

Print files remotely by sending them to a Telegram bot.

This project allows you to send **PDF files or images** to a Telegram bot and have them **printed automatically** on a Windows computer.

---

## What this is

- A **self-serve printing tool**
- Files are sent via **Telegram**
- Printing happens on a **Windows PC**
- Designed for **personal or small office use**

---

## Supported file types

✅ PDF  
✅ Images (JPG / PNG → converted to PDF)

❌ Word, Excel, AutoCAD (DWG), or other formats  
(These must be exported to PDF before sending.)

---

## Requirements

- Windows 10 / 11
- A working printer installed on Windows
- Internet connection
- Telegram account

---

## Installation

1. Download the installer from the release section
2. Install the application
3. Select your printer
4. Start the agent (runs in system tray)

After installation, the agent can start automatically with Windows.
